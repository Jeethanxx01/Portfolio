Microservice E-Commerce Platform

October-December 2024

This project showcases a scalable, microservice-based e-commerce platform with JWT role-based authentication, built using Spring Boot, Eureka Server, API Gateway for backend and React for frontend. It features services with specialized databases: SQL for authentication, MongoDB for product data and availability, and Redis for caching cart data. Apache Kafka is integrated for real-time inventory and stock updates. Order management is handled by a dedicated service. The backend architecture follows OOP principles and leverages JpaRepository and MongoRepository for efficient data handling and ELK stack for server logging.
